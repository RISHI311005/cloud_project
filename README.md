# Food Waste Prediction and Smart Redistribution System

A simple, production-style starter layout for a college cloud ML project.

## Quick start

1. Create a virtual environment.
2. Install dependencies: `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and adjust values.
4. Run the API: `python -m src.main`

## Project structure

See folders under `src/`, `data/`, `models/`, and `docker/` for the core components.
